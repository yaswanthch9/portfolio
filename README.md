# ✅ TaskFlow — Kanban Task Manager

A fully functional Kanban-style task manager built with vanilla HTML, CSS, and JavaScript. No frameworks, no backend — just clean frontend code.

## 🚀 Features

- **Kanban Board** — To Do, In Progress, Done columns
- **Drag & Drop** — Move tasks between columns by dragging
- **Priority Levels** — High, Medium, Low with color indicators
- **Search & Filter** — Real-time search + priority filters
- **Progress Tracker** — Live sidebar stats and progress bar
- **Due Dates** — With overdue highlighting
- **Local Storage** — Tasks persist across browser sessions
- **Responsive** — Works on desktop and mobile
