# 🎓 Student Portfolio — Alex Reyes

A clean, modern, and fully responsive personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks. No dependencies. Just clean code.

## ✨ Features

- **Responsive Design** — Works on all screen sizes
- **Smooth Scroll Reveal** — Sections animate in as you scroll
- **Active Nav Highlighting** — Nav updates based on scroll position
- **Parallax Hero Effect** — Subtle depth in the hero section
- **Mobile Hamburger Menu** — Collapsible nav for mobile
- **No Build Tools** — Just open `index.html` and go

## 📁 Project Structure

```
portfolio/
├── index.html        # Main HTML file
├── css/
│   └── style.css     # All styles
├── js/
│   └── main.js       # Interactivity & animations
└── README.md         # This file
```

## 🚀 Getting Started

### Option 1 — Open locally
Just double-click `index.html` in your file manager, or open it in your browser.

### Option 2 — Live Server (recommended for dev)
If you use VS Code, install the **Live Server** extension, right-click `index.html`, and select **"Open with Live Server"**.

## 🛠️ Customization

1. **Your Name** — Replace `Alex Reyes` in `index.html`
2. **Your Info** — Update About section, email, and social links
3. **Your Projects** — Edit the project cards in the Projects section
4. **Your Skills** — Update the skills section tags
5. **Colors** — Edit CSS variables at the top of `style.css`

## 🌐 Deploy for Free

### GitHub Pages (easiest)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://yourusername.github.io/portfolio`

### Netlify
1. Drag and drop the folder at [netlify.com/drop](https://app.netlify.com/drop)
